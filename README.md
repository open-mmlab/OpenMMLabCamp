### 2.2. MMSIG 日常任务

所有的任务均会发布在[OpenMMLabCamp仓库的Discussion栏](https://github.com/open-mmlab/OpenMMLabCamp/discussions)，贡献者通过在任务详情中评论领取任务，完成任务后提交Pull Request（PR）并在评论区回复PR链接，待PR评审通过后即可获得积分奖励。

详细操作请查看[贡献者操作指南](activity/taskdoc.md)。
